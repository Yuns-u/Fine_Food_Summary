# Fine_Food_Summary
kaggle의 'Amazon-fine-food-reviews' 데이터를 활용한 Abstractive Summary를 실행해본 deep-learning 프로젝트
2021/11/10-2021/11/11 

# 주의
해당 레포지토리는 https://wikidocs.net/72820 의 모델을 통해 생성요약을 실습하고 자신의 프로젝트로 발전시키기 위한 실습장임.
